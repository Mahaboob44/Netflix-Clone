# Netflix Clone

## Overview
This is a responsive Netflix Clone developed using HTML and CSS. The project replicates the Netflix homepage with a modern and user-friendly interface.

## Features
- Responsive design
- Navigation bar
- Hero banner
- Movie sections
- Footer
- Clean and modern UI

## Technologies Used
- HTML5
- CSS3
